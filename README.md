# The Brain Games Project / Проект "Игры разума"
### Description / Описание
This is the first project in the Java developer training. It is a set of five console games for quick wit and logic. When you click on the link, you will see a video demonstration of each game:

1. [Even Game](https://asciinema.org/a/KJX15KQSqqY4h0ut)

2. [Calc Game](https://asciinema.org/a/9Du2w4rRTQjFEnYU)

3. [GCD Game](https://asciinema.org/a/LofxLcJ4C53aPrRW)

4. [Progression Game](https://asciinema.org/a/pjEvrf1MTgHR3z7C)

5. [Prime Game](https://asciinema.org/a/bBipAPqJtAiszM2b)

Это первый проект в рамках обучения профессии Java-разработчик. Он представляет из себя пять консольных игр на сообразительность и логику. При нажатии на ссылку Вы увидите видеодемонстрацию каждой игры:

1. [Проверка на чётность](https://asciinema.org/a/KJX15KQSqqY4h0ut)

2. [Калькулятор](https://asciinema.org/a/9Du2w4rRTQjFEnYU)

3. [Наибольший общий делитель (НОД)](https://asciinema.org/a/LofxLcJ4C53aPrRW)

4. [Арифметическая прогрессия](https://asciinema.org/a/pjEvrf1MTgHR3z7C)

5. [Простое ли число?](https://asciinema.org/a/bBipAPqJtAiszM2b)

### Launch Instructions: / Инструкция по запуску:
1. Clone the repository using the command
   ```
   git clone git@github.com:Artem-Maksimenko09/java-project-61.git
   ```
2. Build the project using the command
   ```
   ./gradlew build 
   ```
3. Launch the game using the command
   ```
   make run-dist
   ```
4. Enjoy the game and train your brain!
<br>

1. Склонируйте репозиторий при помощи команды

